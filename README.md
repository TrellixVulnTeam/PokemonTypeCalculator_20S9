# Pokemon Type Calculator

## Introduction

A Pokemon Type Calculator that shows the strengths/weaknesses of different type combinations and gives you a rating for your specific team.

## Technologies

- Node JS
- React JS
- Javascript

## Running

```sh
npm start
```
